# ONI Tools

**[▶ Open Live Site](https://zheavyofficial.github.io/oni-tools/)**

Fan-made calculator site for *Oxygen Not Included*. Browser-based, no install required. Not affiliated with Klei Entertainment.

---

## Current Tools

### Volcano Timer
Calculates the OFF/ON durations for a conveyor loader attached to a metal volcano.

- **Avg Output** — enter a direct g/s override, or switch to eruption parameters (rate, active time, cycle length) for a calculated average
- **Timer** — outputs OFF and ON seconds, throughput per game cycle, and timer efficiency
- **Steam Room** — calculates how many ice tempshift plates are needed based on room dimensions and target pressure

Supports all metal volcano types: Gold, Copper, Iron, Cobalt, Aluminium, Niobium, Tungsten, Magma, and Minor Magma.

## Planned Tools

1. **Geyser / Vent Timer** — same timer concept for water geysers and gas vents, accounting for multi-cycle dormancy periods
2. **Heat Exchange Calculator** — cooling/heating requirements for steam rooms and aquatuner setups
3. **Seed Tracker** — track all geysers, vents, and volcanoes on a single map seed with taming status and total resource output
4. **Power Grid Calculator** — production vs consumption for a base or isolated grid
5. **Pipe / Conveyor Throughput** — max throughput for liquid/gas pipes and conveyor belts
