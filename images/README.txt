HOW TO DOWNLOAD FROM WIKI:
1. Go to the wiki page listed above
2. Click the image on the page
3. Right-click the full-size image -> Save Image As
4. Rename to the filename shown above (lowercase)
5. Place in this images/ folder

The HTML will use these images if present, and falls back to the
built-in SVG icons if the files are not found.
